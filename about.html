To-Do List Project

This To-Do List is the main project for Week 5-8. I improved the features, cleaned the layout and made the app save tasks inside localStorage so they stay after refreshing the page.

Main Features

  • Add tasks
  • Mark as done or undo
  • Delete tasks
  • Clear all tasks
  • Tasks saved using localStorage

What I Learned

I learned how to update the DOM with JavaScript, how to store data in the browser, and how to make simple features work together in a complete project.
