Project Overview

This To-Do List is my final project for Week 5-8. I improved the app, added more JavaScript features and fixed the design to make it cleaner.

Technologies Used

  • HTML – structure of the pages
  • CSS – layout and simple styling
  • JavaScript – app logic and saving tasks

How the App Works

When I add a task, it gets added to a list using JavaScript. I can mark it done, delete it, or clear everything. The tasks are saved in the browser using localStorage so they stay after refreshing the page.

What I Focused on in Week 5-8

  • Saving and loading tasks from localStorage
  • Cleaning the layout and spacing
  • Testing the app on phone and laptop
  • Writing this README and preparing a short presentation
